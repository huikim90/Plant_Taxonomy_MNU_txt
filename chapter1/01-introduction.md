# 분류학 연구의 전제

모든 분류학 연구는 두 가지 기본 전제를 기반으로 한다.

## 기본 전제
- 자연에는 **구별 가능한 단위**가 존재한다 (예: 종)
- 이러한 단위 사이에는 **논리적 관계**가 존재한다

## 인간과 분류
인간은 일상생활에서도 자연스럽게 분류 활동을 수행한다.

예
- 음식 종류 구분
- 동식물 이름 인식
- 물건의 특징 설명
- 집단 간 관계 판단

## 식물분류학의 중요성
식물은 인간 생활에서 중요한 자원이다.

주요 기능
- 식량 공급
- 의약 자원
- 에너지 자원
- 산소 공급
- 환경 형성

또한 다양한 생물학 분야에서 분류학 정보가 필요하다.
