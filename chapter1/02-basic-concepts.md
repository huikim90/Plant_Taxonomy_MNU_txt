# 계통분류학의 기본 개념과 용어

계통분류학은 식물의 다양성과 진화적 관계를 연구하는 학문이다.

## 주요 용어

### 용어 (Terminology)
특정 개념을 표현하기 위해 사용되는 학술적 단어

### 개념 (Concept)
관찰과 이론을 포함하는 정신적 구성

예
- 종
- 변이
- 분류

### 정의 (Definition)
개념의 의미를 설명하는 공식적인 진술

정의 기준
- 구조적 정의
- 기능적 정의
- 발달적 정의

### 체계 (System)
여러 요소가 통합된 구조

분류학 체계 요소
- 분류
- 식별
- 기재
- 명명
