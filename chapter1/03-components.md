# 분류학의 기본 구성 요소

분류학의 핵심 활동은 네 가지로 구성된다.

- 유별 (Classification)
- 식별 (Identification)
- 기재 (Description)
- 명명법 (Nomenclature)

## 유별 (Classification)

유사성 또는 계통 관계에 따라 생물을 배열하는 과정

분류 체계 유형
- 인위적 분류
- 전형질적 분류
- 계통학적 분류

### 분류군 변경
- 분할
- 통합
- 이전
- 순위 변경

## 식별 (Identification)
미확인 표본을 기존 분류 체계와 비교하여 이름을 결정하는 과정

도구
- 이분법 검색표
- 컴퓨터 기반 키

## 기재 (Description)
분류군의 특징을 기록하는 과정

단계
- 형질 선택
- 형질 상태 결정
- 특징 기록

## 명명법 (Nomenclature)
분류군에 이름을 부여하는 규칙 체계

학명 구조
속명 + 종소명
